# Configuration

Trinity Launcher provides various global configuration options to customize your experience.

## General Settings

You can access the global settings by clicking on the **Settings** icon in the sidebar.

- **Language:** Change the interface language.
- **Theme:** Switch between Light, Dark, or System theme.
- **Data Location:** Manage where your instances and assets are stored.

## Account Management

Manage your Mojang and Microsoft accounts securely.

- **Add Account:** Log in with a new account.
- **Switch Account:** Quickly switch between active sessions.
